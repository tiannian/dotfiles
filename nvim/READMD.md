# Neovim

## Keymappings

### Nvim Tree

- `.e`: Open tree

### Buffer Line

- `<TAB>`: Next buffer
- `<S-TAB>`: Prev buffer

### Telescope

- `.ff`: File
- `.fb`: Buffer
- `.fg`: Grep
- `.fh`: Help Tag
- `.ft`: TODO

### Search

- `/`: Search content
- `n`: Next search
- `N`: Prev search

### TODO

- `tn`: Next TODO
- `tN`: Prev TODO

### Comment

- `./`: Toggle comment
