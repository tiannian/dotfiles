# Neovim

## Keymappings

### Nvim Tree

- `.e`: Open tree

### Buffer Line

- `<TAB>`: Next buffer
- `<S-TAB>`: Prev buffer

### Telescope

- `.ff`: File
- `.fb`: Buffer
- `.fg`: Grep
- `.fh`: Help Tag
- `.ft`: TODO
- `.fn`: Notify

### Search

- `/`: Search content
- `n`: Next search
- `N`: Prev search

### TODO

- `tn`: Next TODO
- `tN`: Prev TODO

### Comment

- `./`: Toggle comment

### Trouble

## Menu

Use `.m` to open Menu.

### Basic

- Format code
- List Buffer
- List Grep
- List File
- Open Tree
- List TODO
- List Notify
- Undo tree

### Rust

- Runnables
- Expand Macros
- Open Cargo.toml
